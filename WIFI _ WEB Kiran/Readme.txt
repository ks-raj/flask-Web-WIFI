# WiFi Profiles Web App

This web app allows users to view WiFi profiles along with their passwords in a secure manner.

## Build by
Kiran Singh - [LinkedIn Profile](https://www.linkedin.com/in/kiran-singh)

## About the Project

The WiFi Profiles Web App is a simple yet powerful tool for managing and accessing WiFi profiles securely. It provides a user-friendly interface for users to view their saved WiFi profiles and passwords.

### Features
- Displays WiFi profiles along with passwords
- Allows users to toggle password visibility for added security
- Built with HTML, CSS, JavaScript, and Python Flask framework
- Uses Tailwind CSS for modern and responsive design

## Build Instructions

To run the WiFi Profiles Web App locally:

1. Clone this repository to your local machine.
2. Install Python if you haven't already.
3. Navigate to the project directory in your terminal.
4. Install the required Python dependencies:
    ```
    pip install -r requirements.txt
    ```
5. Run the Flask app:
    ```
    python app.py
    ```
6. Open your web browser and go to `http://localhost:5000` to view the app.
